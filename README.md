# llm-trace

Distributed tracing for LLM pipelines in a single C++ header.

## Features
- RAII ActiveSpan: auto-records end time on scope exit
- Span hierarchy with parent/child relationships
- Token counts, cost tracking, error recording per span
- Text tree export and OTLP-compatible JSON export
- `LLM_TRACE(name)` macro for zero-boilerplate tracing
- Zero dependencies, C++17, namespace `llm`

## Quick Start
```cpp
#define LLM_TRACE_IMPLEMENTATION
#include "llm_trace.hpp"

{
    LLM_TRACE("pipeline");
    // ... your code ...
}
std::cout << llm::Tracer::global().export_text();
```
